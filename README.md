# MorseNode
Virtual Web Morse Code Station 

Node.js app written in JavaScript and HTML.

GETTING STARTED
===============
1. Install Node.js runtime on machine. (https://nodejs.org/)
2. Clone or download this repo
2. `npm install`
3. `node app.js`
